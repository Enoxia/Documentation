The tool [Kubenum](https://github.com/Orange-Cyberdefense/kubenum) is a command-line tool designed to perform enumeration on a Kubernetes cluster from within a Pod. kubenum do not need any permission on the cluster, indeed, it is built for penetration testers to help them enumerate Pods and Services without using the kube-api and the Pod's ServiceAccount (which is not granted any permission 99% of the time). It primarily focuses on making DNS requests to retrieve information about Pods and Services.
